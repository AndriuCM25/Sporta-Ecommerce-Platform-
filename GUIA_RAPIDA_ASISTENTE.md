## 🚀 Cómo Usar

### **Para Clientes:**

1. **Abrir el chat:**
   - Haz clic en el botón flotante ✨ en la esquina inferior derecha

2. **Hacer una pregunta:**
   - Escribe tu pregunta (ej: "¿Cómo compro?")
   - Presiona Enter o clic en el botón de enviar

3. **Usar sugerencias:**
   - Al abrir el chat verás 4 sugerencias rápidas
   - Haz clic en cualquiera para obtener respuesta inmediata

4. **Cerrar el chat:**
   - Haz clic en la X del header
   - O clic nuevamente en el botón flotante

### **Para Administradores:**

1. **Acceder:**
   - Inicia sesión como admin
   - Ve al panel de administración
   - El botón flotante 📚 aparecerá automáticamente

2. **Consultar el manual:**
   - Pregunta sobre cualquier funcionalidad
   - Usa sugerencias como "Crear producto" o "Ver manual completo"

3. **Resolver problemas:**
   - Describe el problema (ej: "No puedo subir imagen")
   - El asistente te dará pasos para solucionarlo

```
🛍️ Cómo realizar una compra:
```

### **Ejemplo 2: Admin - Crear Producto**

**👤 Usuario:** "Crear producto"

**🤖 Asistente:**
```
📦 Gestión de Productos:

Crear producto:
1. Ve a "Productos"
2. Clic en "+ Nuevo producto"

   - Stock, badge, descripción

4. Marca como "destacado" si quieres
5. Guarda

Editar/Eliminar:
• Usa los botones en cada fila
• Los cambios son inmediatos

¿Necesitas ayuda con algún producto?
```

## 🔧 Personalización

### **Agregar Nuevas Preguntas**

Edita `frontend/src/components/AIAssistant.jsx`:

```javascript
// En knowledgeBase.keywords
'nueva_pregunta|sinonimo1|sinonimo2': {
  response: '🎯 **Título:**\n\nRespuesta aquí...'
}
```

### **Cambiar Colores**

```javascript
// Botón flotante
background: 'linear-gradient(135deg, #TU_COLOR, #TU_COLOR_2)'
```

### **Modificar Sugerencias**

```javascript
const quickSuggestions = mode === 'customer' ? [
  'Tu sugerencia 1',
  'Tu sugerencia 2',
  'Tu sugerencia 3',
  'Tu sugerencia 4'
] : [...]
```
